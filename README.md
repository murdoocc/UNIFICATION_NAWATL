# UNIFICATION_NAWATL
We present the Perl v5 version of the orthographic unifier program. which seeks to unify the different dialectal variants of Nahuatl into a single one.
* The code it takes input from standard input and produces output to standard output.
* The rules are not necessarily applied in the numbered order.
* The regexp can easily be adapted to other programming languages.
* The g switch on the rules s/pattern/substitution/g was omitted, but it should be included to replace all occurrences of the pattern
